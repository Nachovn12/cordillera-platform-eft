CORDILLERA PLATFORM - REPOSITORIOS EXAMEN FINAL TRANSVERSAL (EFT)
Asignatura: Desarrollo Full Stack III - DSY1106
Sección: 001D
Equipo: Grupo Cordillera - Valeria y cia.

Descripción general:
Cordillera Platform implementa una arquitectura distribuida (Polyrepo) de microservicios. A continuación se listan los repositorios individuales de la solución:

1. Repositorio de Orquestación e Infraestructura
URL: https://github.com/Nachovn12/cordillera-platform-eft.git
Descripción: Repositorio central que contiene el Docker Compose para levantar la plataforma completa, toda la documentación técnica (docs/), el arquetipo Maven y la colección Postman (api-rest/).

2. Frontend
URL: https://github.com/Nachovn12/cordillera-frontend.git
Descripción: Interfaz ejecutiva desarrollada en React 19 + Vite.

3. BFF Gateway
URL: https://github.com/Nachovn12/cordillera-bff-gateway.git
Descripción: Backend For Frontend que centraliza la comunicación hacia los microservicios usando Java 25 y Spring Boot. (Puerto 8081).

4. Data Service
URL: https://github.com/Nachovn12/cordillera-data-service.git
Descripción: Microservicio de gestión de datos operacionales, base de datos `data_db`. (Puerto 8083).

5. KPI Service
URL: https://github.com/Nachovn12/cordillera-kpi-service.git
Descripción: Microservicio de cálculo de indicadores, base de datos `kpi_db`. (Puerto 8084).

6. Report Service
URL: https://github.com/Nachovn12/cordillera-report-service.git
Descripción: Microservicio de gestión y exportación de reportes, base de datos `report_db`. (Puerto 8085).
